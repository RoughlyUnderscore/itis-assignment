# Contributing to Student Performance Analyzer

Спасибо за интерес к проекту! Пожалуйста, следуйте этим правилам:

## Как начать
1. Форкните репозиторий
2. Создайте ветку для изменений: `git checkout -b feature/my-feature`
3. Пишите чистый и читаемый код
4. Добавьте тесты для новых функций

## Pull Request
- Описание изменений
- Убедитесь, что `pytest` проходит
- Следите за `flake8`/`black`

Спасибо за вклад!
